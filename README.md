# hair-model-files

used in numerous paper by various authors,
copied from
[www.cemyuksel.com/research/hairmodels](https://www.cemyuksel.com/research/hairmodels):

  + File size: 14.3 MB
  + Strand count: 50,000

  + Vertex count: 2,450,000

+ [wCurly.hair](wCurly.zip)

+ [wWavyThin.hair](wWavyThin.zip)
